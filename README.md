# pest-disease-tracker
A beginner Python command-line project for logging and analyzing plant pest and disease observations.

## Why I Built This

I built this project to combine my background in horticulture with the Python and data analysis skills I am currently developing. The goal is to create a simple tool that could help track plant health issues over time.

## What It Does

The tracker allows a user to:

- Add a plant health observation
- Record the plant name, pest/disease issue, severity, and date
- View all observations
- Search observations by plant name
- Analyze observations for basic patterns

## Features

- Menu-based command-line interface
- Stores observations using Python lists and dictionaries
- Search function for finding records by plant
- Basic analysis including:
  - Total number of observations
  - Average severity
  - Most common issue
  - Most severe observation

## Skills Practiced

- Python functions
- Loops
- Conditional logic
- Lists and dictionaries
- User input
- Basic data analysis logic
- Debugging

## Example Use Case

A gardener, grower, or plant care worker could use this type of tool to track recurring pest and disease issues across different plants.
